Crop Yield Prediction

Project Description

This project is a simple web-based Crop Yield Prediction application using HTML, CSS, and JavaScript.

The application allows users to enter agricultural information such as:

- Rainfall
- Temperature
- Soil pH

The system calculates an estimated crop yield based on the provided inputs.

Note: This is a demo prediction system. The current calculation uses a simple formula and is not a trained machine learning model.

Features

- Simple and user-friendly interface
- Crop selection option
- Input fields for agricultural parameters
- Instant yield estimation
- Responsive design

Technologies Used

- HTML5
- CSS3
- JavaScript
